Cloning this repo use SSH

```
git clone git@github.com:IrvanAhmadP/Toko-Minuman.git
```

```
git clone https://github.com/IrvanAhmadP/Toko-Minuman.git
```

Open the directory:

```
cd Toko-Minuman
```
