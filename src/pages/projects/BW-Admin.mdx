## Getting Started

Cloning this repo use SSH

```
git clone git@github.com:IrvanAhmadP/BW-Admin.git
```

Cloning this repo use HTTPS:

```
git clone https://github.com/IrvanAhmadP/BW-Admin.git
```

Open the directory:

```
cd BW-Admin
```
