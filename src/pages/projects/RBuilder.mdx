## Getting Started

Cloning this repo use SSH

```
git clone git@github.com:IrvanAhmadP/RBuilder.git
```

Cloning this repo use HTTPS:

```
git clone https://github.com/IrvanAhmadP/RBuilder.git
```

Open the directory:

```
cd RBuilder
```

## Installing

First time you need to install all of package json.

```
npm i
# or
yarn
# or
pnpm i
```

## Running Project

To run the project, you can use the command below:

```
npm run start
# or
yarn start
# or
pnpm start
```
